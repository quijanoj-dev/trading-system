# Trading System Framework Pack

This pack is designed to support your workflow:

- NotebookLM as the knowledge brain
- Claude Opus 4.6 in Antigravity as the reasoning layer
- TradingView for coding and backtesting
- TradeZella for validation

## Suggested Order
1. Fill out the trading profile and current system files
2. Normalize paid courses with the extraction template
3. Add external strategies only after they are converted into the same format
4. Use the prompt packs in NotebookLM and Claude
5. Score candidate systems
6. Validate before any automation planning
