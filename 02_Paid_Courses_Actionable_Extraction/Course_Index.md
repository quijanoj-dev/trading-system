## How to Add a Course

Use the `/extract-course` skill in Claude Code. It will:
1. Read the System Extraction Template
2. Process your course material
3. Create a numbered extraction file (e.g., `Course_01_ICT_Mentorship.md`)
4. Update this index automatically
