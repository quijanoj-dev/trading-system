# NotebookLM Prompt Pack

## Prompt 1 — Paid Courses Extraction
Extract only the actionable trading content from these sources.

For each setup, provide:
- setup name
- market and timeframe
- setup definition
- preconditions
- entry conditions
- stop / invalidation
- exit logic
- session filters
- market regime filters
- risk model
- whether the rules are mechanical, semi-mechanical, or discretionary
- screenshots/examples mentioned
- performance claims
- evidence quality
- ambiguities or contradictions

Ignore storytelling, mindset advice, motivational content, and theory unless it directly changes execution.

## Prompt 2 — External Strategy Ranking
Based only on these sources, rank the top strategies most compatible with ES/NQ 1-minute intraday scalping.

## Prompt 3 — Current System Analysis
Analyze my current trading setup and indicators.

## Prompt 4 — Candidate System Design
Using only the sources provided, design 3 candidate systems for my profile:
1. trend continuation system
2. reversal / liquidity sweep system
3. breakout / opening drive system

## Prompt 5 — Version 1 Selection
Rank the 3 candidate systems and tell me which one should become Version 1.0 first.
