# 13-Week Roadmap

**System:** ICT-SMC PO3-AMD — Silver Bullet V1
**Goal:** Pass Apex evaluation by executing A+ setups only, with discipline and strict risk management.

---

## Phase 1 — System Migration (Weeks 1–2)

**Goal:** Build the TradingView execution environment. No live or paper trading required.

### Week 1 — Platform Foundation
- [ ] Inventory all ThinkorSwim indicators currently in use
- [ ] Classify each: must-have / nice-to-have / remove
- [ ] Convert or source Pine Script replacements for core indicators:
  - Market structure tools (BOS/CHoCH/FVG/OB/Stop Hunt)
  - HTF POI tools (OTE, PO3)
  - SMT divergence (confirmed variant only)
- [ ] Create TradingView layouts: execution layout, HTF bias layout, replay layout
- [ ] Validate each indicator using Bar Replay

**Success:** TradingView system is functional and organized.

### Week 2 — System Validation
- [ ] Complete all remaining indicator conversions/replacements
- [ ] Finalize alerts, templates, and execution rules
- [ ] Replay test ≥ 10 historical sessions
- [ ] Apply 6-gate checklist during replay
- [ ] Prepare TradeZella tagging structure

**Success:** TradingView system is final. Ready to begin structured backtesting.

**Rule:** Do not trade the system until you trust the system.

---

## Phase 2 — Backtesting + Paper Trading (Weeks 3–6)

**Goal:** Turn the system into trust through repetition, data, and discipline.

### Week 3 — Structured Backtesting
- [ ] Replay ≥ 5 London or NY sessions
- [ ] Apply 6-gate checklist to every replayed setup
- [ ] Log all results in TradeZella (tag: Backtest)
- [ ] Observe only — no optimization yet

**Success:** 5+ replay sessions. Clear understanding of when setups work and fail.

### Week 4 — Backtest → Paper Transition
- [ ] Complete ≥ 5 additional replay sessions
- [ ] Begin paper trading A+ setups only (max 1–2 trades per session)
- [ ] Log every trade AND no-trade in TradeZella

**Success:** 10+ total backtest sessions. 5+ paper trades. Zero impulsive entries.

### Week 5 — Paper Trading Consistency
- [ ] Paper trade A+ setups only
- [ ] Maintain RR ≥ 1:3 discipline
- [ ] Track win rate and rule compliance %
- [ ] Identify top recurring mistake. Make one rule adjustment max.

**Success:** 10+ paper trades. ≥ 80% rule compliance. Emotional neutrality improving.

### Week 6 — Evaluation Readiness
- [ ] Paper trade as if live: same size rules, same loss limits
- [ ] Review TradeZella stats: drawdown control, RR distribution
- [ ] Decision: Ready / Not Ready for Apex

**Success:** 20+ paper trades logged. Rule breaks minimal or zero. Clear confidence in execution.

---

## Phase 3 — Apex Evaluation + Scaling (Weeks 7–13)

**Goal:** Pass Apex evaluation by executing A+ setups only — not by forcing profits.

### Week 7 — Evaluation Start (Control)
- [ ] Trade smaller, slower, and cleaner than paper
- [ ] Respect daily loss + trailing drawdown at all times
- [ ] Max 1–2 A+ trades per session
- [ ] Journal immediately after each trade
- [ ] No revenge trades. No scaling up. No "just one more."

**Success:** No rule breaks. Account survives intact.

### Week 8 — Consistency Under Pressure
- [ ] Maintain A+ only execution
- [ ] Protect green days (stop after target)
- [ ] Accept red days without recovery attempts

**Success:** Drawdowns shallow. Fewer trades, better quality.

### Week 9 — Process Over P&L
- [ ] Follow checklist without exception
- [ ] Stop trading once daily plan is complete
- [ ] Journal emotions + decisions clearly

**Success:** "No trade" days feel normal. Losses don't trigger impulsive behavior.

### Week 10 — Micro-Scaling (Only if Earned)
- [ ] Increase size incrementally — never emotionally
- [ ] Keep RR ≥ 1:3
- [ ] Monitor emotional response to size change

**Gate:** Only scale if Weeks 7–9 showed zero rule breaks.

### Week 11 — Capital Protection Phase
- [ ] Stop trading early on green days
- [ ] Reduce size after drawdowns
- [ ] Review mistakes daily

### Week 12 — Passing Mindset
- [ ] No deviation from rules
- [ ] No forced trades to "finish faster"
- [ ] Journal as if already funded

### Week 13 — Funded Account Transition
- [ ] Trade funded account exactly like the evaluation
- [ ] Same rules, same size, same checklist
- [ ] No celebration trading

---

## Day 90 Success Benchmark

All boxes must be checked before scaling:

**System:**
- [ ] TradingView execution system is stable and unchanged
- [ ] 6-gate checklist is internalized and automatic

**Process:**
- [ ] Trade only A+ setups
- [ ] Respect "no trade" days
- [ ] Journal every session

**Risk:**
- [ ] RR ≥ 1:3 respected — no exceptions
- [ ] Daily loss rules never violated
- [ ] Max account risk ≤ 10%

**Data (TradeZella):**
- [ ] Controlled drawdowns
- [ ] Fewer rule breaks over time
- [ ] Repeatable setup performance

**Identity:**
- [ ] Act like a professional trader
- [ ] Trust execution over outcomes
- [ ] Ready to scale responsibly

**If all checked → move forward. If not → fix the weakest gate before scaling.**
