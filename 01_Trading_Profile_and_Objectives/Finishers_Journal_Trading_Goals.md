# Finisher’s Journal Trading Goals

## 13-Week Objective
Build and validate one robust ES/NQ 1-minute trading system that is clear, repeatable, and realistic for future automation.

## Why This Matters
- pass funding evaluations consistently
- reduce chaos and discretion
- build confidence in one real edge
- create a path to scale

## Success Criteria
- one chosen System V1
- fully documented rules
- backtest-ready logic
- forward test plan
- clear risk rules
- clear no-trade rules
- validated execution checklist
